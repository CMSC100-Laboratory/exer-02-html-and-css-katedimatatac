Nojo Cafe Management System

A cloud cafe management system built using JavaFX && CSS. it simulates cloud data storage by using file handling in Java to show how data is saved and retrieved, similar to how online cloud databases work. The system includes shared features for cash transactions, table monitoring, menu management, inventory, and community rewards, showing how different modules can work together and use the same data like a real cloud-based system.

Greenvale City Pollution Reduction Planner

This project models a city-wide pollution reduction problem using numerical and symbolic computation through R and RShiny. It simulates how a city can plan and optimize pollution control strategies by selecting and combining different mitigation projects. The system allows users to choose from various pollution reduction options, set reduction targets, and compute the most cost-effective solution. By applying optimization techniques, the program shows how mathematical models can support real-world environmental decision-making.

Strings

A terminal-based replica of NYT Games' Strands built using Python. The project highlights file persistence, user data handling, and creative use of terminal colors and layouts, all within a text-based interface.
